Revision Truncate! 

== Description ==

Delete old revisions from posts.
You specify how many revs are best and delete the rest.

For example:  Keep only the last 10 revisions of any post.

== Installation ==

1. Download the Plugin Revision Truncate! 
2. Unzip file
3. Upload the folder "revision-truncate" into your /wp-content/plugins directory
4. Activate Plug-In
5. Configure the Plugin under tools
5. Have fun with Revision Truncate! :)
